### Инициализируйте базу данных PostgreSQL любым удобным способом
Установка `PostgreSQL` на виртуальную машину под управлением `Debian 13`
https://www.postgresql.org/download/linux/debian/
```bash
sudo apt install -y postgresql-common
sudo /usr/share/postgresql-common/pgdg/apt.postgresql.org.sh
sudo apt install postgresql-18
```

### Загрузите тестовый датасет в PostgreSQL.
```bash
wget https://edu.postgrespro.com/demo-medium-en.zip
gunzip -c demo-medium-en.zip | sudo -u postgres psql
```

### На стороне ClickHouse создайте таблицу для интеграции с PostgreSQL через движок Postgres.
```sql

(
    `aircraft_code` String,
    `seat_no` String,
    `fare_conditions` String
```
